# Bridge Contracts

## Description
Module that contains versions of [bridge-contracts](https://github.com/Bridgeless-Project/bridgeless-contracts).

## Version 1 
The original version of the bridge contracts.

## Version 2
Introduced `ReferralId` tracking to the contracts.

## Version 3 
The third version is implemented to support token swaping. 

Service currently supports all 3 versions of contracts.
